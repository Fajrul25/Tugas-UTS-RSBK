Instalasi
Local Server
•	Import el.war file ke dalam Eclipse
•	Jalankan Server Payara/Tomcat (pada tugas kami menggunakan Tomcat karena payara error)
•	Klik kanan pada project >> Run As >> Run on server
•	Pada browser ketik: http://localhost:8080/el/
8080 tergantung port yang digunakan
